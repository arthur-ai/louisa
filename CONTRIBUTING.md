# Contributing to Louisa

We welcome contributions! Here's how to get started:

1. Fork the repo and clone it locally
2. Create a branch for your change
3. Make your changes and test locally with `vercel dev`
4. Submit a pull request with a clear description

## Development Setup

1. Copy `.env.local.example` to `.env.local` and fill in your credentials
2. Run `npm install`
3. Run `vercel dev` to start the local development server

## Code Style

- ES modules (`import`/`export`)
- No external dependencies — use native `fetch` and `node:crypto`
- Keep functions focused and well-documented

## Questions?

Open a Discussion or reach out via Issues.
